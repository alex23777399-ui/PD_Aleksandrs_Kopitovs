# PD_Aleksandrs_Kopitovs 
Nosaukums: saskaitišana masu gan izmaksu 
 apraksts: Nepieciešams aprēķināt, cik daudz cementa, smilšu un ūdens vajadzes, lai liet betona plātni un saskaiti kopējās izmaksu(ja zināma cementa un smilšu)
Ar attiecibā 1:3:0,5 kur
1 daļa cementa 
3 daļas smilšu 
0,5 daļas ūdens 
Procesā rodas 5% masas zudums 
 Palaišanas instrukcija:
Atver programmu.
Ievadi nepieciešamo betona kopējo masu (kg).
Ievadi cementa cenu par 1 kg.
Ievadi smilšu cenu par 1 kg.
Programma aprēķinās:
Cementa daudzumu (kg)
Smilšu daudzumu (kg)
Ūdens daudzumu (kg)
Kopējās izmaksas
